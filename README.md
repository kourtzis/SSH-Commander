A web application to batch execute commands and scripts to devices via SSH. Initially built with Mikrotik devices in mind but it works with any SSH-enabled device.

Supports:
- Multiple users with admin and operator roles
- Devices list
- Device groups with nesting
- Code snippets with joining
- Flexible job definition with single device and device group targets and target deduplication
- Immediate or scheduled job execution
- Real-time job status update
- Job history with execution stats
- SSH log storage and vieing per target device

...More to come.
